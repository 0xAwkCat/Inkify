# 函数的概念

## 教学目标

- 理解函数的定义：定义域、值域、对应法则三要素
- 掌握函数的三种表示方法：解析式、列表、图像
- 能够判断一个对应关系是否构成函数

## 重点难点

**重点**：函数三要素的理解与应用

**难点**：从"变量依赖关系"到"集合映射"视角的转变；区分函数与一般对应关系

## 教学过程

### 一、导入（5 分钟）

提出问题：

> 一辆汽车匀速行驶，速度 60 km/h。行驶时间 *t* 与路程 *s* 之间有什么关系？

写出关系式 $s = 60t$，引导学生思考：当 *t* 确定，*s* 是否唯一确定？

### 二、新课讲授（25 分钟）

**函数的定义**

设 $A$、$B$ 为两个非空数集。若对 $A$ 中的**每一个**元素 $x$，按某个对应法则 $f$，在 $B$ 中都有**唯一**一个元素 $y$ 与之对应，则称 $f$ 为从 $A$ 到 $B$ 的**函数**，记作：

```
f: A → B
```

其中 $x$ 称为**自变量**，$y = f(x)$ 称为**因变量**。

**三要素**

| 要素 | 说明 |
|------|------|
| 定义域 | 自变量 $x$ 的取值范围（集合 $A$） |
| 值域 | 因变量 $y$ 实际取到的所有值的集合 |
| 对应法则 | $x$ 映射到 $y$ 的规则 $f$ |

**判断函数的两个关键**

1. 定义域中**每个** $x$ 都要有对应的 $y$（不能遗漏）
2. 每个 $x$ 对应的 $y$ 必须**唯一**（一对多不是函数）

### 三、例题（10 分钟）

判断下列对应关系是否为函数：

1. $f(x) = \sqrt{x}$，定义域为 $\mathbb{R}$
2. $f(x) = \sqrt{x}$，定义域为 $[0, +\infty)$
3. $x^2 + y^2 = 1$（$x$ 为自变量，$y$ 为因变量）

### 四、练习（5 分钟）

1. 求函数 $f(x) = \dfrac{1}{\sqrt{x-1}}$ 的定义域
2. 已知 $f(x) = 2x + 1$，求 $f(3)$ 和 $f(a+1)$
