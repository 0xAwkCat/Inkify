## 列表

### 无序列表

- 函数的三要素：定义域、值域、对应法则
- 每个自变量必须有唯一的因变量与之对应
- 两个函数相等当且仅当定义域和对应法则完全相同
- 值域是因变量实际取到的集合，不一定等于陪域

## 行内代码与代码块

函数定义域的求法可以归纳为三类：当分母含变量时令分母 `≠ 0`，当含根式时令 `被开方数 ≥ 0`，当含对数时令 `真数 > 0`。

```python
def is_function(mapping: dict) -> bool:
    """
    判断一个映射关系是否构成函数。
    函数要求：每个 x 对应唯一的 y。
    """
    seen = {}
    for x, y in mapping.items():
        if x in seen and seen[x] != y:
            return False
        seen[x] = y
    return True
```

```js
// 用 JavaScript 计算导数的数值近似
function derivative(f, x, h = 1e-7) {
  return (f(x + h) - f(x - h)) / (2 * h)
}

const f = x => x ** 2
console.log(derivative(f, 3)) // ≈ 6
```

## 数学公式

### 行内公式

设函数 $f(x) = x^2 + 2x + 1$，其定义域为 $\mathbb{R}$，值域为 $[0, +\infty)$。

当 $\Delta x \to 0$ 时，平均变化率趋近于导数 $f'(x_0)$。

### 块级公式

导数的定义：

$$f'(x_0) = \lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}$$

二次方程求根公式：

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

积分的定义（黎曼和）：

$$\int_a^b f(x)\,dx = \lim_{n \to \infty} \sum_{i=1}^{n} f(x_i^*)\,\Delta x$$
