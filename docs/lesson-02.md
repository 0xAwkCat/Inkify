# 导数入门

## 教学目标

- 理解平均变化率与瞬时变化率的区别
- 掌握导数的定义：极限形式
- 能用定义求简单函数的导数

## 重点难点

**重点**：从平均变化率过渡到瞬时变化率的极限思想

**难点**：极限概念的直觉建立；导数定义式的推导过程

## 教学过程

### 一、导入（8 分钟）

**场景引入**

汽车从 A 到 B 行驶了 120 km，用时 2 小时，则平均速度为：

$$v_{平均} = \frac{120}{2} = 60 \text{ km/h}$$

问题：某时刻的**瞬时速度**该如何计算？

引导学生思考：把时间段 $\Delta t$ 不断缩小，平均速度趋近的值就是瞬时速度。

### 二、平均变化率（10 分钟）

设函数 $y = f(x)$，当自变量从 $x_0$ 变化到 $x_0 + \Delta x$ 时：

- 自变量增量：$\Delta x$
- 函数增量：$\Delta y = f(x_0 + \Delta x) - f(x_0)$
- **平均变化率**：

$$\frac{\Delta y}{\Delta x} = \frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}$$

几何意义：割线的斜率。

### 三、导数的定义（12 分钟）

当 $\Delta x \to 0$ 时，若平均变化率趋近于某个确定的值，称该值为函数 $f(x)$ 在 $x_0$ 处的**导数**：

$$f'(x_0) = \lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}$$

几何意义：切线的斜率。

**例：求 $f(x) = x^2$ 在 $x = 2$ 处的导数**

```
Δy/Δx = [(2+Δx)² - 4] / Δx
       = [4 + 4Δx + (Δx)² - 4] / Δx
       = 4 + Δx
```

当 $\Delta x \to 0$，得 $f'(2) = 4$。

### 四、常见导数公式（5 分钟）

| 函数 | 导数 |
|------|------|
| $f(x) = c$（常数） | $f'(x) = 0$ |
| $f(x) = x^n$ | $f'(x) = nx^{n-1}$ |
| $f(x) = \sin x$ | $f'(x) = \cos x$ |
| $f(x) = e^x$ | $f'(x) = e^x$ |

### 五、练习

1. 用定义求 $f(x) = 3x + 1$ 的导数
2. 利用公式求 $f(x) = x^3 - 2x^2 + 5$ 的导数
